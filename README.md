# アプリ概要
## Letter Boxについて
顧客が気軽に従業員に感謝を伝えることができるアプリ。  

## 本リポジトリについて
本リポジトリはフロントエンドのみを担当。  

フロントエンドは客側と管理画面の2つに分かれる。
本リポジトリは「客側」。  

LetteBoxで用いる他のリポジトリは以下の通り。

# 作成目的
「接客態度、気持ちよかったから感謝伝えたいな」と思っても、  
連絡手段が電話や問い合わせフォームしかないことが多い。  
それらを使って伝えるのは仰々しいと思い感謝の気持ちを伝えれないことが多い。  
気軽に伝える手段があると良いなと思い開発。
